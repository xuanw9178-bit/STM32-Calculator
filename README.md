# 基于STM32的简易计算器

## 项目概述
基于STM32单片机开发的简易计算终端，实现了数字输入、四则运算、清除与异常处理功能。

## 硬件连接

| 外设 | 接口 | 说明 |
|-----|------|------|
| 4×4矩阵键盘 | GPIO（PB0-PB7） | 数字0-9、运算符、功能键 |
| 显示模块 | GPIO/FSMC | LCD/数码管显示 |

### 键盘布局
[7] [8] [9] [÷] [4] [5] [6] [×] [1] [2] [3] [-] [0] [.] [CE] [+]


## 软件功能
- 数字输入：0~9、小数点，支持整数与两位小数
- 四则运算：+、-、×、÷
- 显示：实时显示输入表达式与运算结果
- 清除功能：CE（清除当前输入）、AC（全部清除）
- 异常处理：除零检测、数据溢出检测与提示
