基于STM32的简易计算器

项目简介 基于STM32单片机开发的简易计算终端，实现了数字输入、四则运算和异常处理功能。

硬件平台

主控芯片：STM32
输入：矩阵键盘（0-9、小数点、运算符）
显示：LCD/数码管显示屏
软件环境

开发环境：Keil uVision5
编程语言：C语言
实现功能

支持0~9数字、小数点及"+-×÷"四则运算输入
支持整数及两位小数运算，保证计算精度
实时显示运算结果与表达式
CE（当前清除）/AC（全部清除）按键功能
除零、数据溢出等异常检测与提示机制
个人贡献

独立完成按键扫描与四则运算逻辑开发
实现清除、退格功能及异常处理
完成整机调试与bug修复，优化按键响应与系统稳定性
