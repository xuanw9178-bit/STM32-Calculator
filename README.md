# STM32 智能小车 - 基于STM32F103的运动控制系统

## 项目概述
基于STM32F103微控制器开发的智能小车，实现了电机驱动控制、PWM调速和红外循迹功能。

## 硬件连接

| 外设模块 | STM32引脚 | 功能 |
|---------|----------|------|
| L298N IN1-IN4 | GPIO（PA0-PA3） | 电机正反转控制 |
| 定时器PWM通道 | TIM2_CH1（PA0） | 左轮调速 |
| 定时器PWM通道 | TIM2_CH2（PA1） | 右轮调速 |
| 红外循迹传感器（左） | GPIO（PB0） EXTI0 | 循迹信号检测 |
| 红外循迹传感器（右） | GPIO（PB1） EXTI1 | 循迹信号检测 |
| LED指示灯 | GPIO（PC0） | 状态指示 |
| 蜂鸣器 | GPIO（PC1） | 报警输出 |

## 软件结构

SmartCar/ ├── User/ │ ├── main.c # 主函数，初始化与主循环 │ ├── motor.c / motor.h # 电机驱动模块 │ ├── tracking.c / tracking.h # 循迹算法模块 │ ├── led.c / led.h # LED指示模块 │ └── beep.c / beep.h # 蜂鸣器模块 ├── STM32F10x_FWLib/ # 标准外设库 └── Project/ └── SmartCar.uvproj # Keil uVision5工程文件


## 实现功能
- GPIO控制L298N驱动模块，实现前进、后退、转向、停止
- 定时器PWM输出调节电机转速
- 外部中断实时检测红外循迹传感器信号
- 多路传感器数据融合判断路径偏移，差速控制循迹
- 循迹精度：直线段偏差≤2cm，弯道稳定不脱轨
- LED/蜂鸣器多模块协同控制与状态指示

## 调试记录
- 问题1：红外传感器信号抖动 → 增加软件消抖延时
- 问题2：弯道循迹偏离 → 将单路检测改为双路差分检测
- 问题3：电机启动抖动 → 调整PWM占空比渐变策略
